<h2>Como rodar o código?</h2>


<p>
    Para que o código PHP funcione, é necessário possuir o ambiente de desenvolvimento preparado para tal. Para isto, é necessário possuir o PHP devidamente instalado na máquina, e é necessário que o servidor Apache esteja rodando devidamente.
</p>

<p>
    Após se certificar que o servidor Apache está rodando, execute o seguinte comando no terminal, tendo ciência de que, atraveś do terminal, você esteja dentro do diretório correto: <bold>php -S localhost:8000</bold>.
</p>

<p>
    Após rodar o comando, acesse o browser e digite no campo de busca: localhost:8000
</p>
